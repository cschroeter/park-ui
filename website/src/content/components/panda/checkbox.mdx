## Examples

### Indeterminate

The `indeterminate` state signifies a checkbox with only some of its related options selected.

```tsx live
<Checkbox checked="indeterminate" />
```

### With a different color

By default, the checkbox uses the `accent` color from the theme. To render a checkbox with a different color, use the
`colorPalette` prop.

```tsx live
<Checkbox colorPalette="red" defaultChecked />
```
