## Examples

### Range Slider

To use the slider as a range slider, you need to provide an array of values.

```tsx live multi
<Slider.Root min={0} max={100} defaultValue={[33, 66]}>
  <Slider.Label>Label</Slider.Label>
  <Slider.Control>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb index={0} />
    <Slider.Thumb index={1} />
  </Slider.Control>
</Slider.Root>
```

### Unlabeled

The slider can also be used without a label while still showing indicators within the track.

```tsx live multi
<Slider.Root min={0} max={100} defaultValue={[33]}>
  <Slider.Label>Label</Slider.Label>
  <Slider.Control>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb index={0} />
  </Slider.Control>
  <Slider.MarkerGroup>
    <Slider.Marker value={25} />
    <Slider.Marker value={50} />
    <Slider.Marker value={75} />
  </Slider.MarkerGroup>
</Slider.Root>
```
